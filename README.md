# Weather App 🌤️

A simple and responsive Weather Application that provides real-time weather information for any city. The application fetches weather data using an API and displays details like temperature, weather conditions, humidity, and wind speed.

## Features ✨

- Search weather details by city name
- Displays current temperature and weather conditions
- Shows humidity and wind speed information
- Responsive and user-friendly interface
- Real-time weather updates using API integration

## Technologies Used 🛠️

- HTML5
- CSS3
- JavaScript
- Weather API
